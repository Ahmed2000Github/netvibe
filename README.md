NetVibe is a simple Flutter app designed to test internet connection speeds (download and upload). The app measures and displays the user's current network speed in real-time.

## Installation

Install my-project with npm

```bash
  cd netvibe
  flutter clean
  flutter pub get
  flutter run
```
